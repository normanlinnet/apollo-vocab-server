# apollo-vocab-server

## Used Tech

- GraphQL
- Apollo server
- Sequelize

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

## Features

1. Response 100 vocabs for the vocab list
2. Response the detail for each vocab request
3. Find or Create user by email
4. Add/remove vocabs on user's favorite list
